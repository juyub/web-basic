<HR>
  
- W 키를 눌러 점프할 수 있습니다. 점프하여 몬스터를 피하면 +100점
- D 키를 눌러 미사일을 발사할 수 있습니다. 미사일로 몬스터를 보내주면 +100점
  미사일은 293px의 사정거리를 가집니다.
- 몬스터는 매번 랜덤한 속도록 등장합니다.
- 몬스터에게 잡히면 GameOver 

<HR>  
  
- START 버튼을 클릭하여 게임을 시작합니다.
- 몬스터에게 잡히면 재시작 버튼을 클릭하여 처음화면으로 돌아옵니다.
